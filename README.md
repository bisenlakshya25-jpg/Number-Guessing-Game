🎯 Number Guessing Game

A simple console-based Number Guessing Game built using Python. The player has to guess a randomly generated number based on the selected difficulty level.

🚀 Features

* 🎮 Menu-driven interface
* 📖 Game rules section
* 🟢 Three difficulty levels:
    * Easy (1–50)
    * Medium (1–100)
    * Hard (1–500)
* ✅ Input validation
* 🔢 Attempt counter
* 💬 Hint system (Too High / Too Low)
* ⭐ Performance message based on the number of attempts
* 🔄 Play Again option
* 🚪 Quit option

🛠️ Concepts Used

* Variables
* Conditional Statements (if, elif, else)
* Loops (while)
* Functions
* Function Parameters
* User Input
* Random Number Generation (random module)
* Input Validation
* Modular Programming

▶️ How to Run

1. Clone this repository.
2. Make sure Python 3 is installed.
3. Run the following command:

python number_guess.py
